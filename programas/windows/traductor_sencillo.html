Introducción

Esta es una herramienta que te permite traducir a una gran variedad de idiomas, echa específicamente para que esté optimizada con lectores de pantalla, pero esto no te impide a ti, persona sin ninguna discapacidad, de disfrutar de todo su potencial.

¿Cómo nace este programa?

Este programa nace de una necesidad mía para traducir textos de una manera sencilla, y pensando en aquellas personas que no utilizan el lector de pantalla NVDA (el cual tiene este complemento) y por si este también sea declarado como incompatible, tener una alternativa más.
